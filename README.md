# Project Overview

FinTrack is a full-stack web application that helps users manage their personal finances by tracking income and expenses. The system allows users to categorize transactions, set spending limits, and visualize financial data through interactive charts.

# Project Goals

The main goal of this project is to build a full-stack web application that demonstrates:

* Frontend–backend interaction via REST API
* User authentication with JWT
* CRUD operations for financial data
* Data visualization with charts
* Clean and intuitive user interface

---

# Tech Stack

## Frontend

* Angular 17+ (TypeScript)
* HTML, CSS
* Angular Router
* HttpClient

## Backend

* Django
* Django REST Framework (DRF)

## Database

* SQLite (development)
* PostgreSQL (optional)

## Authentication

* JWT (JSON Web Token)

# Core Features

## Expense & Income Tracking

* Add income and expenses
* Edit and delete transactions
* View full transaction history
* Quick input with minimal fields

---

## Categories

* Predefined categories:

  * Food
  * Transport
  * Entertainment
  * Shopping
  * Bills

* Custom category creation

* Filter transactions by category

---

## Data Visualization

* Charts for:

  * Expenses by category
  * Monthly spending
  * Income vs expenses

* Interactive graphs

* Dashboard overview

---

## Budget Limits

* Set spending limits per category
* Monthly budget tracking
* Notifications when limits are exceeded
* Visual indicators (progress bars)

---

## Authentication

* User registration:

  * Name
  * Email
  * Password

* Login with JWT token

* Secure access to personal financial data

---

## Personal Dashboard

* Total balance overview
* Monthly summary
* Top spending categories
* Recent transactions

---

## Financial Insights

* Spending trends over time
* Most expensive categories
* Basic recommendations:

  * “You spend too much on food”
  * “You are within budget”
